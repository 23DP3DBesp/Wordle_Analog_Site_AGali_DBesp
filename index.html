About the System

Game Concept

Wordle Analog is a Java console implementation of the popular Wordle game, where players guess a hidden 5-letter word within six attempts.

Technical Details

Built using pure Java with no external dependencies. Features color-coded feedback, word validation, and score tracking.

Development

Created as a student project by A. Galilejevs and D. Bespalovs using modern programming practices and clean code principles.

User Guide

1. Installation

Download the .zip archive unzip it to a convenient location. Then open the folder and run start.bat

2. Main Menu

Starting, you'll see a menu with options:

  • 1. Start Game — begins a new Wordle session.
  • 2. Statistics — shows your total wins and past games.
  • 3. Rules — displays how to play.
  • 4. Change Nickname — set your custom player name.
  • 5. Exit

Select an option by typing a number (1–5), then press Enter. To confirm a selected option, just press Enter again.

3. Gameplay

In the game, guess a hidden 5-letter word within 6 tries.

After each guess, you'll receive feedback for each letter:

  • Green — correct letter in correct position
  • Yellow — correct letter in wrong position
  • Gray — letter not in the word
